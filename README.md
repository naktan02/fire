# 화재 감지 및 대피 유도 시스템

이 프로젝트는 라즈베리 파이와 아두이노를 활용하여 화재 발생 시 최단 탈출 경로를 안내하는 시스템의 서버 측 구현입니다.

## 기능

- **불 감지**: 웹캠을 통해 빨간색 종이(불)를 인식합니다.
- **탈출구 감지**: 녹색 종이(탈출구)를 인식합니다.
- **경로 탐색**: A\* 알고리즘을 사용하여 불을 피해 탈출구로 가는 최단 경로를 계산합니다.
- **시각화**: 화면에 경로와 방향 화살표를 표시합니다.

## 실행 방법

1. `uv` 설치 (아직 설치하지 않은 경우).
   pip install uv
2. 프로젝트 실행:
   ```bash
   uv run src/main.py
   ```
